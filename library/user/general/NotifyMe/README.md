
# NotifyMe

**NotifyMe** turns your WiFi Pineapple Pager into a more proper pager, giving you the ability to set an alert and inform you about upcoming meetings etc.

- **Author:** 0i41E
- **Version:** 1.0

## Usage

1. Run the payload on the Pineapple Pager and select the desired time for your notification (YYYY-MM-DD HH:MM)

The current time (+1 minute) is selected as default.
